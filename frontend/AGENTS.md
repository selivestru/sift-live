# AGENTS.md

## Monorepo Structure

```
frontend/        React 19 + Vite 8 + TypeScript 6 (this package)
backend/         NestJS 11 + Prisma + Apollo GraphQL (port 5000)
media-server/    Docker compose: MediaMTX, MinIO, Postgres 16, Redis 7
```

## Key Quirks

- **Vite `~` alias** — defined in both `vite.config.ts` (resolve.alias) and `tsconfig.app.json` (paths). Vite's `resolve.tsconfigPaths` does NOT work because tsconfig uses project references. Both must be kept in sync.
- **GraphQL endpoint**: `http://localhost:5000/graphql`. Backend uses `setGlobalPrefix('api')` but it does NOT affect GraphQL.
- **Auth**: JWT in `localStorage` under `accessToken`. Apollo Client has a custom error link that auto-refreshes on `UNAUTHENTICATED`.
- **Socket.IO**: Reconnects on auth token change. Access via `useSocket()` hook (uses `useSyncExternalStore`, not context).
- **Theme**: Class-based dark mode (`.dark` on `<html>`). `init-theme.ts` applies class synchronously before React render to prevent flash.
- **Env validation**: `src/shared/config/env.ts` — `@t3-oss/env-core` + Zod. Prefix `VITE_`. `.env` is required.
- **Codegen** (`bun run codegen`): Requires `backend/src/schema.gql` to exist. The schema is auto-generated by NestJS on backend startup.
- **shadcn**: `style: base-nova` — uses `@base-ui/react` (not Radix). Component aliases: `~/shared/ui`, `~/shared/lib`, `~/shared/hooks`.
- **React Compiler**: Enabled via Babel plugin in vite.config. Do NOT add `useMemo`/`useCallback`/`memo` manually.
- **FSD**: Routes in `src/app/routes/` (auto-generated to `routeTree.gen.ts`). `entities/` is currently empty.
- **Backend**: Global `@AuthGuard` on all routes. Prisma ORM, BullMQ queues, MinIO for media storage. Transcoding via FFmpeg (transcoder module).

## Lint & Format Config

Both `oxlint.config.ts` and `oxfmt.config.ts` define all rules. The AGENTS.md system prompt already has the style summary — refer to those files for authoritative config.

**Ignored by linter/formatter**: `dist/`, `node_modules/`, `src/app/routeTree.gen.ts`, `.agents/`, `src/shared/api/graphql/__generated__/`.
